- Each demo agent follows the Agent Contracts in `docs/modules/order-assistant.md`.

- Router outputs selected agents and routing reasons, but does not call the model or create suggestions.
- Specialist agents return only the shared structured JSON envelope.
- Coordinator consumes specialist outputs only; it must not invent findings or apply changes.
