# Issue 15: Excel and Finance Screens

## Context Pack

- `docs/prd.md`
- `docs/modules/excel.md`
- `docs/modules/finance.md`
- `docs/issues/04-finance-quote-profit.md`
- `docs/issues/05-excel-import-export.md`
- `docs/issues/12-web-app-shell.md`

## Goal

Expose Excel import/export and cost/quote/profit tools in the browser.

## Scope

- Upload customer purchase-list template.
- Upload supplier package/logistics template.
- Display import row errors.
- Download key Excel exports.
- Edit Goods Line quote/Target Markup.
- Add order/goods cost and charge lines.
- Show profit summary.

## Acceptance Criteria

- Admin can import and export through web forms.
- Invalid headers and row errors are visible.
- Admin can update Target Markup and see recalculated sales price/profit.
- Warehouse User cannot access finance screens.
