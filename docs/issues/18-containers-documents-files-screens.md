## Goal

Expose loading records, document generation, and file downloads in the browser.

## Scope

- Container plan and loading forms.
- Loading list download.
- Export document blockers.
- Draft/final Commercial Invoice generation.
- Draft/final Packing List generation.
- Version history and download links.
- Local file-byte storage for uploads and generated outputs.

## Acceptance Criteria

- Admin can create containers and loading records.
- Admin can generate documents and download Excel/PDF files.
- Final documents are blocked when required fields are missing.
- Generated document versions are listed.
- Tests cover document/download routes.
