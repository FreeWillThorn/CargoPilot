## Summary

When Task Understanding selects risk prompting, call the live-model Order Risk Agent and show supplier-grouped risk suggestions without creating system change drafts.

## Requirements

- Call the Order Risk Agent only when the user explicitly or clearly implies risk review.
- Include selected Import Order context, goods, supplier grouping, destination/origin information when available, and source summaries.
- Allow model-determined open-ended risk categories.
- Require each risk to show basis, affected goods or source items, Supplier grouping when known, suggested documents or actions, confidence, and review need.
- Support `pending` and `ignored` statuses only.
- Do not generate local keyword fallback risks.
- Do not create system Change Drafts from risk suggestions in MVP.
