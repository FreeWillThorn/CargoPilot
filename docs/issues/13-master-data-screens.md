## Goal

Build Admin screens for Suppliers, Consignees, Warehouses, and System Settings.

## Scope

- List/create/edit Suppliers.
- List/create/edit Consignees.
- List/create/edit Warehouses.
- Edit System Settings.
- Duplicate Supplier warning display.

## Acceptance Criteria

- Admin can manage all master data from browser forms.
- Warehouse User cannot access master data screens.
- Consignee defaults are visible for later Import Order creation.
- Tests cover form submission and permission denial.
