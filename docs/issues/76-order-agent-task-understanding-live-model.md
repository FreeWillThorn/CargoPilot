## Summary

Call the live-model Task Understanding Agent to decide whether a conversation needs data entry, risk prompting, both, neither, or refusal.

## Requirements

- Reuse the existing DeepSeek configuration from 基础资料; do not add a new settings page.
- Send natural-language input, attachment summaries, and optional selected Import Order context to the Task Understanding Agent.
- Require a successful model response before producing any user-facing agent conclusion.
- Store and display the business summary plus collapsible raw structured response.
- Display clear failure when the model is unconfigured, unavailable, times out, or returns unusable output.
- Support out-of-scope or high-risk refusal results.
