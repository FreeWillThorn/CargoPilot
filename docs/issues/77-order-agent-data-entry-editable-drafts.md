## Summary

When Task Understanding selects data entry, call the Data Entry Agent and render editable draft applications without writing system records.

## Requirements

- Call the live-model Data Entry Agent with conversation messages, source summaries, and optional selected Import Order context.
- Support Order Creation Draft, Order Update Draft, Minimal Goods Line Draft, Master Data Draft, and allowed field update drafts.
- Enforce the MVP field allowlist from `docs/modules/order-agent.md`.
- Show unknown model fields as unmapped information, not confirmable fields.
- Allow a Minimal Goods Line Draft when at least a Chinese or English goods name is known.
- Show missing-field questions and move the conversation to `waiting_for_input` when needed.
- Render editable draft forms with scroll containers for long goods lists and wide tables.
