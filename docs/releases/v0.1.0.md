# CargoPilot v0.1.0

First MVP release.

## Included

- Role-based local web app with Admin and Warehouse users.
- Import Order and Goods Line management.
- Supplier, Consignee, Warehouse, and system settings screens.
- Fixed-header Excel import/export workflows.
- Quote, markup, cost, charge, and profit summary tools.
- Warehouse receiving workflow with arrival exceptions and file metadata.
- Dashboard, reminders, global search, and Goods Line tracking filters.
- Container planning, container creation, loading records, and loading list exports.
- Commercial Invoice and Packing List draft/final generation with versioned xlsx/pdf downloads.
- Repeatable demo seed command and release smoke command.

## Verification

Run:

```sh
make smoke
```

Expected result:

- Unit test suite passes.
- Demo seed completes against `/tmp/cargopilot-smoke.sqlite3`.

## Demo

```sh
make seed
make serve
```

Open `http://127.0.0.1:8000`.

Demo logins:

- Admin: `admin@example.com` / `admin`
- Warehouse: `warehouse@example.com` / `warehouse`
